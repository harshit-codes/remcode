# remcode-stdio.js

**File Path:** `bin/remcode-stdio.js`

## Overview

STDIO Bridge for MCP Inspector

This script creates a STDIO interface that bridges to the HTTP MCP server,
allowing the MCP Inspector to work with Remcode's HTTP transport.
